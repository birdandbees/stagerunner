Set all prefixes for child stageRunners.

Usage

stageRunner_.set_prefixes(prefix = "")

Arguments

prefix
character. The prefix to assign to this stageRunner.

Description

When a stageRunner is used in conjunction with an objectdiff::tracked_environment, we need to remember the full nested tree structure. This function sets up the prefix member of each sub-stageRunner recursively to enable correct remembering functionality.
